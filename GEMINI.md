# Alphabet Practice Web App (알파벳 쓰기 연습)

초등학교 3학년을 위한 영어 4선 노트 기반 알파벳 쓰기 연습 및 퀴즈 웹 앱입니다.

## 🚀 프로젝트 개요
- **목적**: 초등학생들이 올바른 획 순서와 4선 노트 위치에 맞춰 알파벳을 쓰는 법을 학습.
- **주요 기능**:
  - **연습하기**: 대/소문자별 가이드(Comic Sans MS)를 보고 직접 쓰기 연습.
  - **획 순서 보기**: 애니메이션과 숫자 뱃지를 통해 쓰기 순서 확인.
  - **퀴즈 풀기**: 가이드 없이 랜덤으로 제시된 글자를 쓰고 실력 점검.
  - **진행 현황**: 학습한 글자들의 상태를 로컬 스토리지를 통해 추적.

## 🛠 기술 스택
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Drawing**: HTML5 Canvas API (Pointer Events 통합)
- **Animation**: SVG + CSS/SMIL Animation
- **State Management**: React Hooks + LocalStorage
- **Fonts**: Comic Sans MS (학습 가이드), Baloo 2 (제목), Nunito (본문)

## 🏃 실행 및 빌드
- **개발 서버 실행**: `npm run dev` (기본 포트 3000)
- **프로덕션 빌드**: `npm run build`
- **프로덕션 실행**: `npm run start`
- **린트 체크**: `npm run lint`

## 📏 핵심 가이드라인 및 컨벤션

### 1. 4선 노트 시스템
캔버스(400x400) 및 SVG(100x100) 내에서 다음 좌표를 기준으로 정렬합니다:
- **L1 (Top)**: 18.5%
- **L2 (Middle)**: 39.5%
- **L3 (Baseline/Red)**: 60.5%
- **L4 (Bottom)**: 81.5%

### 2. 채점 시스템 (MSE 로컬 채점)
- **방식**: 사용자의 획과 레퍼런스 이미지를 픽셀 단위로 비교 (Mean Squared Error).
- **임계값(Threshold)**: **0.32** (아이들의 서툰 필체를 너그럽게 수용하기 위해 상향 조정).
- **특이사항**: 위치와 상관없이 '모양'만 비교하는 **개별 바운딩 박스 크롭** 방식 사용.

### 3. 드로잉 및 피드백 설정
- **선 두께**: **10px** (아이들의 가독성과 필기감을 위해 상향 조정).
- **채점용 내부 두께**: **24px** (폰트 굵기와 매칭 최적화).
- **오답 피드백**: 오답 시 사용자의 획을 **빨간색**으로 표시하고 파란색 가이드(0.3 투명도)와 겹쳐 보여줌.

### 4. 획 데이터 (`lib/strokeData.ts`)
- 모든 소문자는 기하학적으로 완전한 원형과 부드러운 아치형을 지향.
- 'a', 'g', 'q' 등은 시계 반대 방향(CCW)으로 원을 그리도록 설계.

### 5. 퀴즈 진입 조건
- 각 모드(대문자/소문자)별로 최소 **10개 이상**의 글자를 '완료(pass)'해야 퀴즈에 참여 가능.

## 📂 주요 디렉토리 구조
- `/app`: 페이지 라우팅 및 레이아웃
- `/components`: 드로잉 캔버스, 모달, 채점 결과 등 핵심 컴포넌트
- `/lib`: 획 데이터(`strokeData.ts`), 채점 로직(`grading.ts`), 스토리지 유틸(`storage.ts`)
- `/public`: 매니페스트 및 정적 자산
